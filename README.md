# Command Explorer App

The Command Explorer App is a graphical user interface (GUI) application built using Python's `customtkinter` library. It enables users to conveniently explore and retrieve information about various commands along with their syntax and examples.

## Features

- **Search and Select Modes:** The app offers two modes: "Search Mode" allows users to input a specific command, while "Select Mode" offers a dropdown list of commands to choose from.

- **Syntax and Example Display:** Upon entering or selecting a command, the app fetches and displays the corresponding syntax and example details from a Microsoft Access database named `DataBackend.accdb`.

- **Modern UI Design:** The GUI boasts a modern and visually appealing dark-themed design.

## Prerequisites

- Python 3.x
- `customtkinter` library
- Microsoft Access Database Driver

## Usage

1. Install the required dependencies: `pip install customtkinter pyodbc`

2. Run the application: `python app.py`

3. Choose the desired mode and input/select a command to view its syntax and examples.
